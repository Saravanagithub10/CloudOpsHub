# CloudOps Hub

## Overview

CloudOps Hub is a cloud monitoring and operations dashboard built using Microsoft Azure services.

The application provides visibility into system health, service availability, alerts, deployments, and incident tracking through a centralized dashboard.

---

## Features

* System Health Monitoring
* Service Availability Tracking
* Alert Management Dashboard
* Incident Timeline
* Deployment History
* Azure Function API
* Azure Application Insights Monitoring
* GitHub Actions CI/CD
* Azure Static Web App Hosting
